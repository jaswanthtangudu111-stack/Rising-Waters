#  Rising Waters (A Flood Prediction System using Machine Learning)

An AI/ML project developed as part of the **AI-ML-and-GEN-AI Track Project**.
The system predicts flood probability using environmental and historical data through a machine learning model and provides predictions via a Flask web application.

---

##  Repository Structure

```text
1.Brainstorming & Ideation
2.Requirement Analysis
3.Project Design Phase
4.Project Planning Phase
5.Project Development Phase
6.Project Testing
7.Project Documentation
8.Project Demonstration
```

Each folder contains the corresponding deliverables, reports, presentations, and source files for that phase of the project.

---

##  Project Overview

The Flood Prediction System uses machine learning techniques to estimate flood probability based on numerical input features.
The trained model is integrated with a Flask web application to provide an interactive user interface for predictions.

##  Running the Project

Navigate to the development phase folder:

```bash
cd "5.Project Development Phase"
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
