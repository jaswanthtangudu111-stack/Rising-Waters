About Rising Waters

An AI-powered Flood Prediction System using Machine Learning.

🌊 Project Overview

Rising Waters is an intelligent flood prediction system developed using Machine Learning. The application analyzes various environmental, geographical, and infrastructural factors to predict the probability of flood occurrence and assist in early disaster preparedness.

🎯 Objective

To provide an accurate flood prediction model that helps authorities and communities make informed decisions by identifying flood-prone conditions before disasters occur.

📊 Dataset

  • Source: Kaggle
  • Records: 50,000
  • Features: 20
  • Target: Flood Probability

🤖 Machine Learning Model

  • XGBoost Regressor
  • R² Score ≈ 95%
  • MAE ≈ 0.0085
  • RMSE ≈ 0.011

💻 Technology Stack

  • Python
  • Flask
  • XGBoost
  • Pandas
  • NumPy
  • HTML5
  • CSS3
  • Bootstrap 5

🚀 Key Features

  • AI-powered flood prediction
  • Interactive web application
  • Real-time probability estimation
  • Risk categorization
  • Safety recommendations
